# Mapping the Cloud: A Mixed-Methods Study of Cloud Security and Privacy Configuration Challenges

This repository contains the replication package and supplementary material for the paper:

> **Mapping the Cloud: A Mixed-Methods Study of Cloud Security and Privacy Configuration Challenges**  
> To appear at **NDSS 2026**

---

## Repository Contents

- **Replication package**  
  Materials to support replication of the study, including:

  - Scripts and notebooks used for data collection and preprocessing of Stack Overflow posts
  - Topic modeling code and configuration (e.g., LDA training and evaluation)
  - Analysis scripts for mapping topics to use cases and configuration challenges

- **Supplementary material**
